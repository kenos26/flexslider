Flexslider: comment trouver le passage ?

1- Rendez vous à cette adresse:

http://flexslider.woothemes.com/

2-Telechargez le fichier proposé

Bravo ! Vous avez gagné un rigolo

3-Decompressez le fichier téléchargé

4-Renommez le dossier extrait " Flexslider2"

6-Dans le dossier "Flexslider2", trouvez le fichier "flexslider.css". Placez-le dans votre dossier "css"

7-Dans le dossier "Flexslider2", trouvez le fichier "jquery.flexslider.js". Placez-le dans votre dossier "js"

8-Integrez le code ci dessous dans le headla page html ou vous souhaitez integrer le slider:

"link rel="stylesheet" href="css/flexslider.css""

9-Integrez le code ci dessous dans le de la page html ou vous souhaitez integrer le slider:

script src="js/jquery.flexslider.js"

10-Integrez le code ci dessous dans la page html ou vous souhaitez integrer le slider:

$(window).load(function() {

$('.flexslider').flexslider({

animation: "slide"

});

});

11-Creez une div a l'endroit ou vous souhaitez creer votre caroussel. Donnez lui la classe "flexslider"

12-Creez une ul dans la div. Donnez lui la classe "slides"

13-Creez une li dans la ul

14- Creez une img dans la li

15-Activez le gant d'Infinité





15-Et voila, vous avez réussi a créer un slider, malheureusement la moitié de l'univers ne pourras jamais le voir...
